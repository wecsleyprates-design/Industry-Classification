The Most Important Thing to Understand Before Reading Anything

Worth AI already receives all the data it needs for complete global industry classification. OpenCorporates sends uk_sic-62012 for every UK company. Trulioo sends .sicCode next to .naicsCode in the same object. Equifax sends 25 industry columns. ZoomInfo sends SIC alongside NAICS. The jurisdiction_code for 16M+ GB companies sits in Redshift. Historical code changes are in rel_business_industry_naics.

None of this is used. The pipeline reads a fraction of what it receives, applies a deterministic rule to pick one NAICS winner, and discards the rest. The Consensus Engine does not add new data sources — it reads the same data more completely, encodes it as 38 numeric features, and applies a trained XGBoost model to produce calibrated probabilities, multi-taxonomy output, and AML/KYB risk signals.
